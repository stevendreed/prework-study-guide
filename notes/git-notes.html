Git Notes

  • Useful cmds:
    • git [-h | --help] [-a | --all] {cmd} : prints common commands, ALL git cmds, or man page for {cmd}
    • git status : reports branch & commit status
    • git clone {URL} : clone a repository to the working directory
    • git pull origin {branch} : syncs local {branch} clone with remote repository of {branch}, {branch} = main is "rebasing" code
    • git checkout [-b | --build] {branch} : switch to {branch}, -b flag to build new branch with name {branch}
    • git add [-A | --all] : stages changed files for later commit
    • git push [origin] {branch} : apply changes to {branch}
    • git commit [-m | --message] "{imperative commit message}"
    • git revert {hash} : (careful!) version hash to revert branch to
    • git diff --check : lists possible whitespace errors
    • git config [-g | --global] user.email "you@example.com" : configures local git to reference your email when you commit
    • git config [-g | --global] user.name "Your Name" : set name associated with commits from local changes
