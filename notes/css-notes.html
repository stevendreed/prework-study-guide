CSS Notes

  • Cascading Style Sheets (CSS) are companion attributes for HTML elements
  • CSS offers three different scopes to modify an element
    • Inline CSS, which applies to a specific HTML tag using the "style" attribute
    • Internal CSS style sheet, embedded into the head element and modifies any element of that class
    • External CSS style sheet, which is imported with the link tag and contains changes that applies to the whole page
  • A property with a value is called a "declaration"
    • A CSS property has a value which is set, such as {color: blue;}
  • CSS Box Model: Margin{Border{Padding{Content}}}
